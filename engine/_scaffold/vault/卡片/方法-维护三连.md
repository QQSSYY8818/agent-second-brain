# 方法-维护三连

每次改动卡片后跑三连（+三环），是本框架唯一的必修维护动作：lint 保证卡片合规，catalog 再生索引，routemap/rings 刷新检索入口。

1. `python engine/vault_lint.py` — 规范检查，不过就修（`--fix` 可校正时间戳）
2. `python engine/vault_catalog.py` — 再生 总目录/邻接表/现况总览/热核
3. `python engine/vault_routemap.py` — 刷新第 0 跳路由地图
4. `python engine/vault_rings.py` — 刷新三环视图（顺带清过期辅助标）
